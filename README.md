# Arithmetic Expression Interpreter

##  Overview

This project implements an interpreter that evaluates arithmetic expressions using recursive evaluation techniques and abstract syntax trees.

##  Objective

To demonstrate how programming languages interpret and execute expressions internally.

##  Concepts Demonstrated

* Interpreter design
* Expression evaluation
* Abstract syntax trees (AST)
* Recursion
* Functional programming concepts

##  Technologies Used

* Functional programming
* Algorithmic evaluation
